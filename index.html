Arrecife Mobil

App Mobil

Aplicación creada con el objetivo de tener un almacenamiento seguro de credenciales en las cuentas de usuario que se crean en las diferentes plataformas y servicio.

Nos permite guardad Nombre de usuario, contraseña y dirección de correo para cada cuenta registrada. No automatiza el inicio de sesión en los registros, solo permite tener un registro seguro de la información para consultarlo cuando sea necesario.

También permite registrar información textual de manera segura.

Utiliza una encriptación de información que se basa en la contraseña escrita y que solo podrá ser desencriptado al utilizar la App Arrecife Mobil.

Esta App Mobil fue creada utilizando el SDK Flutter con el lenguaje Dart.

Creado en Android Studio

Proyecto SIC (Sistem Inputs Controller)

Desktop App para Sistemas operativos Windows

Sistema que cumple el objetivo de registrar de forma numérica la cantidad de pulsaciones del teclado y del mouse con el objetivo de tener un control sobre el uso de estos dos periféricos.

El sistema incluye un autocliker que no suma clics a los contadores de clics para tener una contabilización limpia del uso del mouse.

Este programa fue creado con el IDE Visual Studio 2022 utilizando Windows Form y C#

También se utilizó Git para el versionamiento del código
